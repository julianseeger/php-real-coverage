php-real-coverage
=================

Calculates the real coverage from an existing PHPUnit/CodeCoverage report

contribute
==========

Execute the tests
```
composer install --prefer-dist --dev
./vendor/bin/phpunit
```

Make your changes tested and in PSR-2

Execute the the tests again

And make your Pull Request
